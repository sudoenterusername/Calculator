Simple calculator app I made while learning how to make native android apps. 
It has different layouts for landscape and portrait. 
Make sure to check out my notes if you want to start learning android studio with java. 
Honestly, kotlin is the recommend language by google but you may need java at some point so it is definitely worth your time. Though, if you want to start building ASAP kotlin makes more sense, especially if you don't know any java.
